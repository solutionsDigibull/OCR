# Skill: Document Understanding with Docstrange

## Description
This skill allows the AI to process, analyze, and extract structured information from documents such as PDFs, images, and text files.

## Capabilities
- Extract text from PDFs and images
- Summarize documents
- Answer questions from documents
- Extract key-value data

## Workflow
1. Read the document
2. Extract text
3. Analyze content
4. Return structured output

## Use Cases
- Invoice extraction
- Resume parsing
- Contract analysis
